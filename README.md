* # 博客系统开发

  

  ## 开发环境

  * IDE: IDEA , VS_CODE 
  * 编程语言: Java , Vue.js , Nuxt.js , Sql语句
  * 数据库: MySql , Redis
  * 技术栈: Spring 全家桶
  * Java版本: jdk 1.8
